## Quickstart

1. Clone this project

   ```shell
   git clone https://github.com/fabiopalacio/twitter.git
   ```

2. Install dependencies:

   ```shell
   cd twitter
   pip install -r requirements.txt
   ```

3. Run local dev server:

   ```shell
    python manage.py migrate
    python manage.py runserver
   ```
